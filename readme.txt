== Description ==

Widget for showing X random Facebook friends. Requires a Facebook account and a Facebook Application set up for your site.

== Installation ==

1. Upload `tiutiu-facebook-friends-widget`-folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. In the `Appearance` -> `Widget` menu, drag and drop the widget to your preferred area
4. Enter your `Application ID` (retrieved from [http://www.facebook.com/#!/developers/apps.php](http://www.facebook.com/#!/developers/apps.php `Facebook | My Application`)) as well as your `Application Secret` in the widget settings.
5. Click the Save button.
6. A Facebook Login button should appear. Hit it!
7. Grant the widget permission to access your data offline.
